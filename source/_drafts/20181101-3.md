# 任务取消

如果外部代码能在某个操作正常之前将其置入“完成”状态，那么这个操作就可以称为可取消的(Cancellable)。

取消某个操作的原因有很多： 

- 用户请求取消
  用户点击图形界面程序中的“取消”按钮，或者通过管理结构来发出取消请求
- 有时间限制的操作*
